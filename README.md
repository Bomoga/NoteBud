### NoteBud

NoteBud is an AI-powered notebook and study companion that creates a smart, personalized
workspace for a student's classes. Unlike generic note-taking apps, NoteBud learns and
adapts through machine learning and continuous feedback, becoming smarter as students use
it.

## Core Features

• Smart Course Notebooks - Each course gets its own dedicated notebook storing
uploaded files, transcripts, notes, and links with RAG-powered Q&A interface.
• Study Environment - Built-in Deep Focus mode, Pomodoro timers, ambient audio,
and friendly interface for long study sessions.
• Trust & Transparency - Answers include citations to specific notebook chunks,
groundedness indicators, and warnings when evidence is insufficient.
• Planner & Review - Study planner surfaces what to review before exams with
spaced repetition using AI-generated review cards.
